# Changelog

All notable changes to this project will be documented in this file.

## 0.0.2

- 保留默认快捷键，同时支持用户通过 VS Code 标准方式自定义绑定
- 优化命令标题，便于在快捷键面板中搜索
- 将 README 调整为中文，并补充快捷键重绑定说明与 `keybindings.json` 示例

## 0.0.1

- Initial release
- Added editor context menu command to copy code fragment location
- Added default shortcut `Ctrl+Shift+C`
- Added support for single and multiple selections
- Added automated tests and VSIX packaging configuration
