# 实现计划

## 当前目标

- 构建一个 VS Code 扩展
- 在编辑器有选区时提供右键菜单“复制代码片段定位”
- 默认快捷键为 `Ctrl+Shift+C`
- 复制内容为当前文件相对路径与选区行号范围

## 当前设计

- 单选区输出：`src/foo.ts:12-18`
- 多选区输出：`src/foo.ts: 12-18 20-21`
- 相对路径基于工作区根目录
- 无选区时不显示右键项，快捷键不生效
- 非工作区文件或非本地文件给出提示，不复制错误内容
